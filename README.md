# Warehouse Assistant Chatbot

## About
AI-powered chatbot for Amazon Fulfillment Center 
associates built using Anthropic's Claude API.

## Problem It Solves
FC associates need quick answers about:
- Picking procedures
- Safety protocols
- Inventory queries
- Warehouse operations

## Built With
- Python
- Anthropic Claude API
- python-dotenv

## How to Run
1. Clone the repository
2. Install: pip install anthropic python-dotenv
3. Add API key to .env file
4. Run: python chatbot.py
